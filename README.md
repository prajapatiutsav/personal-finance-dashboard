# Personal Finance Dashboard

A responsive web dashboard that analyzes personal expenses and predicts future spending.

## Features
- Expense analytics
- Expense prediction
- Interactive charts
- Add expenses
- Responsive UI

## Tech Stack
- Python
- Flask
- Pandas
- NumPy
- Chart.js
- HTML
- CSS
- JavaScript

## How to Run

Install dependencies:

pip install flask pandas numpy

Run the app:

python app.py
